Welcome!

By the end of this project you will know how to:

  • Edit React files

To get started editing, simply click on the CodeSandbox project embedded on the right. The rest of this document describes the tasks required for this project. Create a CodeSandbox account so that all of your Projects get saved in one place when you save changes to the provided templates.

    Deliverable

    The deliverable for this project is a report comparing the salary for your degree major to the highest and lowest majors in the provided data set.

    The report will conatin the following brief sections:

    1. A description of your major and one other that you are interested in
    2. An updated 'Comparing Salaries' chart
    3. An simple analysis comparing your major to the highest and lowest

    Step 1: Look Around and Stay Calm
    Open the original CodeSandbox embeded at the top of the page an look it over. Watch the linked short tutorial. Just start building some familiarity. Stay calm: you don't have to understand any of these things.

    Step 2: Make Changes
    To make a long story short:

    • Open the 'Degrees that Pay' Table
    • Find the higest, lowest, your major, and one other major
    • Open the Project CodeSandbox
    • Update the values (the red arrows above) in the data.js file

    Step 3: Download Chart
    The hard part is done. Congratulations! Simply press the DownloadChart button over the graph. Select Save File and OK. The chart will download automatically to your local Downloads Folder. Now:

    • open the file that was downloaded,
    • right click on the chart,
    • copy and paste it into your report document
    • resize the chart to your liking

    Step 4: Write Your Simple Report
    It does not have to be perfect.
    Save this report as a .pdf file; this all should fit on a single page.
