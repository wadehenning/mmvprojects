Tiny Bubbles

Learning Outcomes: Random Numbers, Customizing Plots, Euclidean Distance, Clustering

Welcome Back!

By the end of this project you will know how to:

  • Generate random numbers
  • Use them to create a Bubble Plot
  • Calculate the Euclidean Distance
  • Use that distance to create a data cluster

Once again, simply click on the CodeSandbox project embedded on the right to begin working on the project. If you have not already, please create a CodeSandbox account so that all of your Projects get saved in one place when you save changes to the provided templates. KEY NOTE: Please use the Firefox web browser if possible (the download chart button does not work on Google Chrome

Bubble plots are very powerful data visualization tools. For a little inspiration, watch the video below. It shows a scientist modeling health and wealth outcomes for 200 countries over 200 years.

  • 200 Countries in 200 Years

The Details

Deliverable

The deliverable for this project is a report containing the updated Bubble Chart. By 'updated', I mean that all of the identified model parameters should have been changed

The report will contain:

  1. The function used to generate the (x,y) locations of the points
  2. A description of "Euclidean Distance"
  3. An updated 'Scatter / Bubble / Cluster' chart
  4. An description of the '200 Countries in 200 Years' analysis

Step 1: You've Done this Before
Open the original CodeSandbox embeded at the top of the page an look it over. Watch the '200 Countries in 200' Years video. After completing this project you have 95% of the tools to create a similar analysis.

Step 2: Make Changes
To make a long story short:

  • Open the CodeSandbox
  • Update all of the parameters identified in the image above.
  • Now a little tricky: open the 'ChartWrapper' file. Near line 25 is the code function that generates the coordinate values. Make sure to include this code in the analysis and give me your best guess on how it works.

Step 3: Review the Chart (Download With Firefix)
The hard part is done. Look over the Bubble Plot and make sure that there is a cluster in the middle of the chart. This cluster was created by calculating the distance that each point is from the center of the graph using the Euclidean Distance. You chose the value of the distance to define a 'cluster'.

  • download the chart (w/ Firefox)
  • (or take a picture)
  • copy and paste it into your report document
  • resize the chart so that it looks nice

Step 4: Write Your Simple Report
It does not have to be perfect.
It does have to include the Euclidean Distance and 200 Countries descriptions.
Save this report as a .pdf file; this all should fit on a single page.
